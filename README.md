
# _Hello, World! (hello)

A hello world feature

## Example Usage

```json
"features": {
    "ghcr.io/devcontainers/feature-starter/hello:1": {
        "version": "latest"
    }
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| greeting | Select a pre-made greeting, or enter your own | string | hey |
